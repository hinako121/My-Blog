  ✨ 功能清单

  📝 博客文章

  - 文章列表 / 详情页（完整 CRUD）
  - 富文本编辑器 — 集成 WangEditor，支持图文混排、代码高亮、引用块
  - 自动标签提取 — 从标题中智能解析 #标签 和 【标签】 两种格式
  - 字数统计 — 自动去除 HTML 标签后精确计数

  📷 相册

  - 图片网格展示 + 灯箱(lightbox)大图预览
  - 支持 ESC 键关闭、点击遮罩关闭
  - 悬停放大动效

  📔 日记

  - 时间轴(timeline)布局 — 视觉化时间线 + 心情 emoji 标记
  - 每条日记可附带 mood 心情标签

  🛠️ 工具分享

  - 卡片式工具推荐，支持分类标签
  - 外部链接跳转

  💬 留言板

  - 游客可匿名留言
  - 字数实时计数
  - 发布后即时刷新（乐观更新）

  🏠 首页

  - 三栏布局：左侧个人卡片 + 标签云 + 最近留言 | 中间文章列表 | 右侧日历 + 站点统计
  - 迷你日历 — 纯前端实现，高亮今天
  - 站点统计：文章数、留言数、总字数、运行天数、标签数

  🔐 后台管理

  - JWT 认证（7天有效期，Bearer Token）
  - 文章管理（增删改）
  - 相册管理（上传+编辑）
  - 日记管理（增删改）
  - 工具管理（增删改）
  - 个人设置（昵称、头像上传）
  - 站点设置（背景图片等）

  🎨 设计系统亮点

  1. 玻璃拟态 (Glassmorphism) 设计语言 — 全套 CSS 变量驱动的磨砂玻璃效果，带渐变高光和内边框，视觉效果很高级
  2. 暗色/亮色主题切换 — 支持 Light / Dark / 跟随系统三种模式，所有颜色通过 CSS 变量统一管理
  3. 响应式设计 — 大屏三栏 → 平板两栏 → 手机单栏，逐步降级
  4. 滚动揭示动画 — IntersectionObserver 驱动的交错入场动画（stagger reveal），支持 prefers-reduced-motion 无障碍
  5. 骨架屏加载态 — shimmer 动画骨架屏，每个列表页都有
  6. 空状态设计 — 每种内容类型都有专门的空状态提示（图标 + 文案）
  7. 自定义字体 — 使用「龚帆乐淘体」作为展示字体
  8. 浮动卡片动画 — 侧边栏卡片带有交错延迟的呼吸浮动效果
  9. 导航栏下划线跟随 — 鼠标悬停/路由激活时底部指示条平滑过渡
  10. 全站背景图片 — 支持从后台设置自定义背景图，带渐变色彩叠加层

  ---
  🎯 总结

  这个项目最大的亮点是设计完成度非常高。作为一个个人博客，它不仅在功能上覆盖了文章、相册、日记、工具分享、留言板等完整模
  块，更在视觉设计上做到了一致且精致的玻璃拟态风格，暗色/亮色主题切换、响应式适配、无障碍支持、动画细节都考虑得很周全。
  代码结构清晰，前后端分离规范，是一个可以作为个人博客生产部署的完整项目。
